# Бот матвея

Телеграм-бот знакомств на aiogram.

## Запуск локально

1. Создай `.env` рядом с кодом:

```
BOT_TOKEN=your_token_here
```

2. Установи зависимости:

```
pip install -r requirements.txt
```

3. Запусти:

```
python ip_bot_code.py
```
